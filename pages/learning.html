There is a lot of information about Python programming online. Official Python documentation and wiki are good places to start, but there are also many other useful resources.

Starting from scratch

  • Learn Python: in-browser interactive tutorial
  • An Hour of Python: somewhat inspired by HOC
  • How to Think Like a Computer Scientist: interactive version of a popular book by the same name
  • CS 61A: The Structure and Interpretation of Computer Programs (in Python)
  • Programming for Everybody: Coursera course
  • Programming Foundations with Python: Udacity course
  • Introduction to Computer Science and Programming Using Python: MIT/EdX course
  • Code Combat: turtles are a bit pass√©
  • CheckIO: gamified exercises
  • exercism: realistic exercies and code review by peers
  • Python for Fun: non-trivial programs and exercises for intermediate programmers
  • newcoder: more advanced tutorials
  • Automate the Boring Stuff with Python
  • Learn Python the Hard Way
  • A Byte of Python and Python for you and me: more free books
  • Python programming fundamentals: nice short tutorial
  • python3.codes: Cool blog with fun code examples

For developers new to Python

  • Lean X in Y minutes: where X == 'Python'
  • Code Like a Pythonista: Idiomatic Python: Zen, PEP8, and much more
  • Python Module of the Week: a tour of the Python standard library through examples
  • The Hitchhiker's Guide to Python: best-practices guide
  • Full Stack Python: web development in depth
  • awesome-python: curated list of Python libraries and projects
  • Python Cookbook, 3d ed: free online version (intermediate -- advanced level)
  • Python 3 Patterns, Recipes and Idioms: collectively created book (intermediate -- advanced level)
  • Python Anti-Patterns

For teachers and educators

  • PythonEDU SIG: mailing list for python in education special interest group
  • PythonEDU WG: mailing list for python education working group
  • Learn to code with me: if unsure where to start
  • Python Tutor: online visualization of code execution with one click "share session" option to allow immediate instructor-student interaction.
  • Invent with Python: teaching through programming games
  • Python Books: collection of free books and guides on specific topics
  • Raspberry Pi: tons of educational materials
  • skulpt: in-browser implementation of Python
  • Reeborg's World: robo-turtles
  • Teaching with the IPython Notebook: nothing's wrong with IDLE except that evolution happens...
  • Software Carpentry: Teaching basic software engineering skills to scientists
